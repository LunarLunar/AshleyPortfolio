# AshleyPortfolio 技術克服與特效解析文件

這份文件記錄了在開發本作品集網站時，所使用的核心特效邏輯與遭遇的技術瓶頸，以及我們是如何克服這些問題的。目的是為了未來若有開發者或您自己遇到類似需求時，能夠快速參考與重用這些思路。

## 一、核心特效設計思路

### 1. Dynamic Portfolio Tiling (動態遞歸鋪磚佈局)
- **原理**：不使用傳統的 CSS Grid 或 Flexbox，而是透過 JavaScript 中的遞歸演算法 (`generateLayout`)，將畫面視為一個矩形，根據作品數量不斷「切半」，產生不同長寬比的矩形區塊座標 (`left`, `top`, `width`, `height`)。
- **優勢**：能夠確保畫面永遠被作品無縫填滿，且沒有任何空白。當每隔 6 秒重新觸發運算時，配合 CSS 的 `transition: all 0.8s`，所有圖塊會自動產生像活物般擠壓變形的流暢動畫。

### 2. LensFX (獨立多組鏡頭微運鏡)
- **原理**：捨棄傳統的 CSS Animation，使用 `requestAnimationFrame` 建立一個每秒 60 幀的高效渲染迴圈 (`lensAnim`)。
- **作法**：在每一幀中，透過 `Math.sin()` 與 `Math.cos()` 將時間轉換為波浪數值，並動態賦予圖片不同的 `transform-origin` 與微小 `scale`，模擬出緩慢平移、縮放、甚至手持晃動的感覺。
- **隔離設計**：每張圖的相位（`phaseOff`）與週期（`lensCycle`）都是亂數生成，確保每張圖片的運鏡軌跡完全獨立，不會有死板的同步感。

### 3. Staggered Fade Wave (空間座標階梯式淡入淡出)
- **原理**：每 30 秒觸發一次的全局重整特效。為了避免所有圖片同時消失顯得生硬，我們透過擷取 DOM 元素的 `getBoundingClientRect()` 取得其在畫面上的實體位置。
- **作法**：透過 `(ra.top + ra.left) - (rb.top + rb.left)` 對所有圖塊進行排序，讓左上角的圖片排在最前面，右下角排在最後面。接著使用迴圈搭配 `setTimeout`（間隔 0.1 秒），創造出如同水波紋般「由左上掃到右下」的退場與進場效果。

## 二、遭遇問題與技術克服

### 🎈 問題一：全局過場動畫時發生「畫面凍結」與「卡頓」
- **狀況描述**：在 30 秒一次的大過場（Fade Out → 黑畫面 → Fade In）啟動前，畫面瞬間凍結，LensFX 運鏡停止，且 Tiling 重新排列時出現掉幀與不順暢。
- **原因分析**：
  1. 為了節省效能，之前的邏輯在開始淡出時就將 LensFX 迴圈提早 Return（暫停計算），導致視覺上圖片瞬間失去動能。
  2. Tiling 在每 6 秒刷新排版時，程式碼不斷將同一張圖片的 `url(...)` 重新塞給元素的 `backgroundImage`。這會觸發瀏覽器無意義的樣式重繪 (Style Recalculation & Repaint)，引發嚴重的卡頓。
- **解決方案**：
  1. 解除 LensFX 的封印，讓其在過場的漸黑漸亮期間持續運作，因為「維持運動慣性」能大幅提升使用者感受到的滑順度。
  2. 在 `syncTilesWithData` 中加入比對防護：只有當 `cam.style.backgroundImage !== newBg` 時才更新背景圖，徹底根除不必要的重繪開銷。

### 🎈 問題二：燈箱 (Lightbox) 內直向圖片被嚴重壓縮
- **狀況描述**：手機上呈現正常（圖片佔大比例），但在電腦大螢幕上，直立圖片因為被固定的上下佈局（Flex Column）限制了高度，導致直立圖片被迫縮得非常小。
- **原因分析**：原本的佈局採用了固定比例（Flex: 4 和 Flex: 1），沒有考量到直向圖片需要釋放垂直空間。
- **解決方案（智能佈局切換）**：
  1. 引入了 JavaScript 的 `Image.onload` 動態維度判定：在圖片讀取完畢瞬間判斷 `naturalHeight > naturalWidth`。
  2. 若為直向長圖，立刻為容器加上 `.is-portrait` 類別。
  3. 透過 CSS Media Query (`min-width: 769px`) 介入：遇到 `.is-portrait` 時，強行將燈箱由垂直排版（Column）切換為水平排版（Row）。
  4. 讓直向圖片在左側無限制擴展高度，並將文字說明固定於右側寬度為 `320px` 的獨立側欄中。這讓系統能具備「辨識圖片方向並自動切換最優佈局」的智能。
